# Arena Billiard

This is a Spring Boot application for managing a Billiard Arena.

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- Vaadin
- H2 Database (Runtime/Dev)
- MySQL (Production)
- Lombok

## Getting Started

1. Clone the repository.
2. Ensure you have Java 21 installed.
3. Run the application using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```

## Development
This project uses Vaadin for the frontend UI and Spring Boot for the backend.
